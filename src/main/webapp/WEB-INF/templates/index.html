Spring email with Thymeleaf

This is a demo application exemplifying the use of Thymeleaf for creating and sending several kinds of HTML email in a Spring application.

Note that, although for educational purposes this is a web application developed with Spring MVC and HTML5, these same techniques would work perfectly in a standalone application with no web interface at all.

In order to run the application to should configure your SMTP server correctly. You can do this by modifying the values on src/main/resources/configuration.properties and src/main/resources/javamail.properties.
